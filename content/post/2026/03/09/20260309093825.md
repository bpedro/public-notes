Many companies build expensive portals that are full of marketing text but lack basic search and testing tools. If a developer cannot find what they need in thirty seconds, they will look elsewhere. A portal that doesn't help you code is a waste of space.

My work in developer experience shows that utility is the only metric that matters. I’d focus on clear search, "try it now" buttons, and code snippets for common languages.
